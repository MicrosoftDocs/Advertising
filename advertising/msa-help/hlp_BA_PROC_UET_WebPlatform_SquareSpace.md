# Set up UET tags using Squarespace

If you created and manage your website with Squarespace, you can add and manage your UET tag there.

## Implementing UET using Squarespace

Before completing the following steps at Squarespace, copy your Microsoft Advertising Universal Event Tracking (UET) tag ID. Select **Conversion Tracking** > **UET tags** and copy the tag ID.

1. Sign in to your Squarespace account and select the webage you would like to track conversions for.

   :::image type="content" source="media/BA_Conc_UET_SquareSpace_Dashboard.png" alt-text="Dashboard in Squarespace.":::
1. Install the UET tag.

   1. Select **Settings**.
   1. Select **Advanced**.
   1. Select **Code Injection**.

      :::image type="content" source="media/BA_Conc_UET_SquareSpace_CodeInjection.png" alt-text="Select Settings, select Advanced, select Code injection.":::
   1. Paste the UET tag in the **HEADER** section.

      :::image type="content" source="media/BA_Conc_UET_SquareSpace_Header.png" alt-text="Paste the UET tag in the HEADER section.":::
   1. Select **Save**.
