# How do I delete a UET tag?

You can delete unused UET tags that aren't associated with conversion goals, audiences or shared with other accounts. Once a UET tag is deleted, it will no longer show in the grid. Don't know what UET is? See [What is UET and how can it help me?](hlp_BA_CONC_UETv2WhatIsTag.md)

> [!IMPORTANT]
> The per account limit for UET tags is 1200. You can't create more UET tags without deleting existing tags once you've reached this limit.

## Delete a UET tag

1. From the navigation menu on the left, hover over **Conversions** and select **UET tag**.
1. Select the checkbox next to the UET tag(s) you want to delete.
1. Select **Delete**.
1. Select **OK** when prompted with **Are you sure you want to delete the selected tag(s)?**

> [!NOTE]
> If a UET tag doesn't meet the requirements for deletion, you will see an error with the number of changes that weren't made. Select **Show errors** to see which conditions weren't met.

> [!IMPORTANT]
> Deleting goals or tags can impact campaign performance, especially if those conversions were used for optimization.

## Remove conversion goals from an existing UET tag

1. From the navigation menu on the left, hover over **Conversions** and select **Conversion goals**.
1. Select the conversion goal you want to modify.
1. Select **Next** until you are on the "Set up tagging" page.
1. Select the **Use a different tag** button.
1. From the dropdown menu, select the UET tag that you would like to use.
1. Select **Save**.
1. Select **Save and Next**.
1. Select **Done**.

> [!NOTE]
> After removing all conversion goals from a UET tag, you can delete the tag by following the steps above.
