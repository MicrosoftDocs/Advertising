# Edit your smart page

> [!NOTE]
> Microsoft Smart Page will no longer be available after **May 31, 2026**. After this date, you won't be able to access existing Smart Pages or create new ones. [Learn more](hlp_BA_CONC_SmartPages.md)

## Edit website components

1. Sign in to Smart Pages.
1. From the top of the page, select **Smart Page builder** > **View, edit, and publish my Smart Page**.
1. Use the **Components** panel on the left side to select the component you'd like to edit. You can also select any component in the website preview to edit it. [Learn more about website components](hlp_BA_CONC_SmartPages.md)
1. When you've finished editing a component, select **Back to components** to edit a different component. You can also select any component in the website preview to edit it.
1. To add or edit optional components, select **Add another component**.
1. Verify the edits in the website preview.
1. In the upper-right corner, select **Publish** to update your website with your edits.
