# Smart campaigns - Code-free conversion goals

> [!NOTE]
> Not everyone has this feature yet. If you don't, don't worry—it's coming soon!

You can use the intelligence of Microsoft Clarity to set up custom, code-free conversion tracking. Enable Clarity within your tracking tag setup and track conversions by event (for same revenue value and for variable revenue value) without any code or scripts.

## Set up a conversion goal with a tracking tag

1. Create a new conversion goal. [Learn more about creating a conversion goal](hlp_BA_CONC_SmartCamps_Intro.md).
1. In the **Set up tagging** section, you can choose a tracking tag or create a new one:

   **Choose an existing tracking tag**

   1. Choose a tracking tag from the **Select your tracking tag** dropdown menu.
   1. Select an option to answer the question **Do you have this tracking tag installed on your website?**
   1. Select **Next**.
   1. Under **Add event tag**, select **Code-free, powered by Microsoft Clarity**.
   1. Select the **Enable Microsoft Clarity** checkbox.

      > [!NOTE]
      > If Clarity is already enabled for this tracking tag, you don't have to do anything here.

      > [!NOTE]
      > To proceed with code-free setup, you must either select a tracking tag or select the **Enable Microsoft Clarity** checkbox.
   1. Select **Next**.

   **Create a tracking tag**

   1. Select either **Install the tag yourself** or **Send the tag to a developer**.
   1. Select **Next**.
   1. Under **Add event tag**, select **Code-free, powered by Microsoft Clarity**.
   1. Select the **Enable Microsoft Clarity** checkbox.

      > [!NOTE]
      > If Clarity is already enabled for this tracking tag, you don't have to do anything here.
   1. Select **Next**.
1. Under **Which URL would you like to open?**, enter the URL for the website you would like to set up your conversion goal on.
1. After you are redirected to your site, select an interaction that you would like to track directly. You can select either a custom event or revenue tile.
1. Fill out the **Your goal details** section.
1. Choose the currency on your page from the dropdown menu.
1. (Optional): Under **Set up your event tracking**, select **Remove** next to an event or revenue item to remove it.
1. Select **Complete** to return to the Microsoft Advertising portal.
