# Make bulk changes to your campaigns and ad groups within Microsoft Advertising

## Make changes across multiple campaigns

1. From the navigation menu on the left, hover over **Campaigns** and select **Campaigns**.
1. Select the checkbox next to the campaigns that you want to edit.
1. Select **Edit** in the blue action bar.
1. You can make these changes across multiple campaigns at once:

   - Campaign status (enable, pause, or delete selected campaigns)
   - Daily budgets
   - Lifetime budgets
   - Bid strategy
   - Campaign language
   - Audience association
   - Autogenerated asset settings
   - Website exclusions
   - IP address exclusions
   - End date
   - Ad schedule time zone

## Exclude websites and IP addresses for campaigns

1. From the navigation menu on the left, hover over **Campaigns** and select **Campaigns**.
1. Select the checkbox next to the campaigns that you want to edit.
1. Select **Edit** in the blue action bar.
1. Select **Other changes**.
1. From the dropdown menu next to **Campaign settings**, select **Website exclusions** or **IP address exclusions**.
1. Enter the the sites or address you'd like to exclude.
1. Select **Apply**.

## Make changes across multiple ad groups

1. From the navigation menu on the left, hover over **Campaigns** and select **Ad groups** > **Ad groups**.
1. Select the checkbox next to the ad groups that you want to edit.
1. Select **Edit** in the blue action bar.
1. You can make these changes across multiple ad groups at once:

   - Ad group status (enable, pause, or delete selected ad groups)
   - Current bids
   - Bid strategy
   - Ad group language
   - Audience association
   - URL options
   - Ad distribution
   - Website exclusions
   - End date
   - Ad schedule time zone

## Make changes to your ad distribution across multiple ad groups

1. From the navigation menu on the left, hover over **Campaigns** and select **Ad groups** > **Ad groups**.
1. Select the checkbox next to the ad groups that you want to edit.
1. Select **Edit** in the blue action bar.
1. Select **Other changes**.
1. From the dropdown menu next to **Select a setting**, choose **Select your ad distribution**.
1. Choose **The entire Microsoft Advertising Network (recommended)** or **Microsoft sites and select traffic**.
1. Select **Apply**.
