# Asset guidelines and accepted file types

## Image guidelines and requirements

- **File formats**: The accepted file types for images are GIF (without animation), JPG, JPEG, and PNG.
- **Minimum file size**: 703 x 368 pixels.
- In the [Ads Studio](hlp_BA_CONC_AdStudio.md), please note that you can add the same image to different folders, but you cannot have duplicate images in the same folder.
- **Recommended image dimensions**: Below are the recommended image dimensions per aspect ratio. To help ensure that your images look the way you want in different ad placements, please follow these recommendations.

  | Aspect ratio | Recommended dimensions (pixels) |
  | --- | --- |
  | **1.91:1** | 1200 x 628 |
  | **1:1** | 1200 x 1200 |
  | **1:2** | 600 x 1200 |
  | **4:1** | 1200 x 300 |
  | **1.5:1** | 1200 x 800 |
  | **1.33:1** | 1200 x 900 |
  | **1.78:1** | 1200 x 674 |
  | **1.72:1** | 1200 x 698 |
  | **1.69:1** | 1200 x 710 |
  | **1.55:1** | 1200 x 774 |
  | **0.93:1** | 1200 x 1290 |
- For information about recommended image dimensions for Audience ads, image extensions, Multimedia ads, and Performance Max campaigns, see [Best practices for Audience ad images](hlp_BA_CONC_ImagesForAudienceAds.md), [Add images to your ad with image extensions](hlp_BA_PROC_AddImageExtension.md), [Multimedia ads](hlp_BA_CONC_MMA_Search.md), and [Create a Performance Max campaign](hlp_BA_PROC_CreatePmax.md).

## Video guidelines and requirements

- **Length**: Must range from six to 120 seconds
- **File formats**: We recommend MPEG-2, MP4, or MOV. We support all [Azure Media Services v3](https://go.microsoft.com/fwlink/?linkid=2175202) input file formats.
- **Aspect ratio**: Must range from 16:9 to 9:16 (e.g., 1:1 is acceptable)
- **Resolution**: As high as possible. At least 720 x 720 pixels recommended, and at least 120 x 120 pixels required.
- **Maximum file size**: 10 GB
- **Source**: Videos must be uploaded from your local computer or from a publicly accessible file server such as Dropbox, FTP, or OneDrive. Indirect links to videos from sites such as Dailymotion, Facebook, Twitch, or YouTube are not allowed.
