# AddHTML5s Service Operation - Campaign Management
Adds array of HTML5 assets to a specified campaign.

::: zone pivot="soap"

## <a name="request"></a>Request Elements
The *AddHTML5sRequest* object defines the [body](#request-body) and [header](#request-header) elements of the service operation request. The elements must be in the same order as shown in the [Request SOAP](#request-soap). 

> [!NOTE]
> Unless otherwise noted below, all request elements are required.

### <a name="request-body"></a>Request Body Elements

|Element|Description|Data Type|
|-----------|---------------|-------------|
|<a name="html5assets"></a>HTML5Assets|Array of HTML5 assets to add to the specified campaign.|[HTML5](html5.md) array|

## <a name="example"></a>Code Syntax
The example syntax can be used with [Bing Ads SDKs](../guides/client-libraries.md). See [Bing Ads API Code Examples](../guides/code-examples.md) for more examples.
```csharp
public async Task<AddHTML5sResponse> AddHTML5sAsync(
	IList<HTML5> hTML5Assets)
{
	var request = new AddHTML5sRequest
	{
		HTML5Assets = hTML5Assets
	};

	return (await CampaignManagementService.CallAsync((s, r) => s.AddHTML5sAsync(r), request));
}
```
```java
static AddHTML5sResponse addHTML5s(
	ArrayOfHTML5 hTML5Assets) throws RemoteException, Exception
{
	AddHTML5sRequest request = new AddHTML5sRequest();

	request.setHTML5Assets(hTML5Assets);

	return CampaignManagementService.getService().addHTML5s(request);
}
```
```php
static function AddHTML5s(
	$hTML5Assets)
{

	$GLOBALS['Proxy'] = $GLOBALS['CampaignManagementProxy'];

	$request = new AddHTML5sRequest();

	$request->HTML5Assets = $hTML5Assets;

	return $GLOBALS['CampaignManagementProxy']->GetService()->AddHTML5s($request);
}
```
```python
response=campaignmanagement_service.AddHTML5s(
	HTML5Assets=HTML5Assets)
```
