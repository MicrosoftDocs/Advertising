# AISearchSetting Data Object - Campaign Management
Defines the AI search setting data object.

## Syntax

# [XML](#tab/xml)

```xml
<xs:complexType name="AISearchSetting" xmlns:xs="http://www.w3.org/2001/XMLSchema">
  <xs:complexContent mixed="false">
    <xs:extension base="tns:Setting">
      <xs:sequence>
        <xs:element minOccurs="0" name="AISearchEnabled" type="xs:boolean" />
        <xs:element minOccurs="0" name="AutoGeneratedImageOptOut" nillable="true" type="xs:boolean" />
        <xs:element minOccurs="0" name="AutoGeneratedTextOptOut" nillable="true" type="xs:boolean" />
        <xs:element minOccurs="0" name="FinalUrlExpansionOptOut" nillable="true" type="xs:boolean" />
        <xs:element minOccurs="0" name="SearchTermMatchingOptOut" nillable="true" type="xs:boolean" />
      </xs:sequence>
    </xs:extension>
  </xs:complexContent>
</xs:complexType>
```

# [JSON](#tab/json)

```json
{
  "Type": "AISearchSetting",
  "AISearchEnabled": "ValueHere",
  "AutoGeneratedImageOptOut": "ValueHere",
  "AutoGeneratedTextOptOut": "ValueHere",
  "FinalUrlExpansionOptOut": "ValueHere",
  "SearchTermMatchingOptOut": "ValueHere"
}
```

## <a name="elements"></a>Elements

The [AISearchSetting](aisearchsetting.md) object has the following elements: [AISearchEnabled](#aisearchenabled), [AutoGeneratedImageOptOut](#autogeneratedimageoptout), [AutoGeneratedTextOptOut](#autogeneratedtextoptout), [FinalUrlExpansionOptOut](#finalurlexpansionoptout), [SearchTermMatchingOptOut](#searchtermmatchingoptout).

|Element|Description|Data Type|
|-----------|---------------|-------------|
|<a name="aisearchenabled"></a>AISearchEnabled|Enable AI search.|**boolean**|
|<a name="autogeneratedimageoptout"></a>AutoGeneratedImageOptOut|Opt out of auto-generated images.|**boolean**|
|<a name="autogeneratedtextoptout"></a>AutoGeneratedTextOptOut|Opt out of auto-generated text.|**boolean**|
|<a name="finalurlexpansionoptout"></a>FinalUrlExpansionOptOut|Opt out of final URL expansion.|**boolean**|
|<a name="searchtermmatchingoptout"></a>SearchTermMatchingOptOut|Opt out of search term matching.|**boolean**|

The [AISearchSetting](aisearchsetting.md) object has [Inherited Elements](#inheritedelements).

## <a name="inheritedelements"></a>Inherited Elements

### <a name="inheritedelementssetting"></a>Inherited Elements from Setting
The [AISearchSetting](aisearchsetting.md) object derives from the [Setting](setting.md) object, and inherits the following elements: [Type](#type). The descriptions below are specific to [AISearchSetting](aisearchsetting.md), and might not apply to other objects that inherit the same elements from the [Setting](setting.md) object.  

|Element|Description|Data Type|
|-----------|---------------|-------------|
|<a name="type"></a>Type|The type of setting.<br/><br/>For more information, see [Setting Data Object Remarks](setting.md#remarks).|**string**|
